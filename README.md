# Arch-i3-Dotfiles

Contains dot files of my Arch+i3-gaps install.
* Distro: Arch Linux
* Window Manager: i3-gaps
* Terminal: Termite (switched from urxvt)
* Shell: zsh 5.3.1
* Bar: Polybar

_I use [wal](https://github.com/dylanaraps/wal) to generate my color scheme_
